Mengenal Semantic Elements Di HTML5

  • Article

Ketika kita ingin membuat layout untuk menampilkan block artikel artikel yang ada di website kita,tag article bisa kita gunakaan dengan element header dan footer.
Karena tag article memiliki type block, artinya element ini memiliki lebar browser jika kita tidak menentukan lebarnya.

  • Aside

Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa manambahkan aside dibagian bawah postingan.
Karena aside memiliki type/sifat block maka lebarnya sesuai jika kita tidak menentukan lebarnya.
